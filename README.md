# abdelrahman.ai

Personal portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design for all devices
- Customizable design system (press `Ctrl+/` or `Cmd+/` to open tweaks panel)
- Enhanced neural network background animation
- Multiple background themes and color options

## Quick Start

Edit `index.html` to change:
- Name, bio, and stats
- Social media links
- Portrait image (`assets/photo.png`)

View live at [abdelrahman.ai](https://abdelrahman.ai)
